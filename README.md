## Data Storage

All data is stored in the browser's `localStorage` under the key `elta_lp`. This means:

- **Data stays on the device** — it does not sync between devices or browsers
- **Clearing browser data deletes the draft** — use Export to keep a permanent copy
- **Storage limit** is typically 5-10MB per origin — more than enough for lesson plans
- **No data is sent to any server** — everything runs locally

## Coaching Context

This tool was designed to address specific patterns observed across 9 ELTA coaching goal sheets:

| Observed Challenge | How the Tool Helps |
|---|---|
| Low Student Talk Time | Live STT gauge forces planning of student output per stage |
| Over-reliance on L1 (Mandarin) | Strategy menu includes TPR, gestures, routine English, word walls |
| Lack of instructional clarity | ICQs, CCQs, visual instructions, I-We-You model as selectable strategies |
| Abrupt lesson closures | Dedicated Homework/Review section with transition field |
| Unclear co-teacher roles | Role dropdowns in both Standard and Simplified modes |
| Activities not aligned to objectives | Objectives section at the top, visible during planning |
| No formative assessment | Strategy menu includes exit tickets, thumbs up/down, traffic lights, observation checklists |
